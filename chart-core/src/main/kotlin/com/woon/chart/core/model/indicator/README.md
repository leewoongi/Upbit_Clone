# Indicator

차트의 지표 및 보조 요소들을 정의합니다.

## 패키지 구조

```
indicator/
├── grid/            # 그리드 (가격 눈금선)
├── crosshair/       # 십자선 (터치 포인트 표시)
└── bollingerband/   # 볼린저 밴드 (기술적 지표)
```

## 설계 원칙

- **데이터만 표현**: 위치, 값, 활성화 상태 등 순수 데이터만 저장
- **색상/스타일 제외**: UI 관련 속성은 chart-ui-android에서 처리
- **불변성**: 상태 변경 시 copy()로 새 인스턴스 생성
